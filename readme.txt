Описание:
Этот скрипт на Python использует библиотеку Selenium для автоматизации задач веб-скрапинга на сайте "https://journal.top-academy.ru". Приложение предназначено для автоматизации процесса входа на сайт, перехода на конкретную страницу для парсинга и извлечения ссылок на iframe из блоков с видео.

Цель:
Цель данного приложения - автоматизировать процесс входа на сайт "https://journal.top-academy.ru", получения доступа к определенной странице и сбора ссылок на iframe из блоков с видео. Собранные ссылки на iframe могут быть использованы для дальнейшего анализа или интеграции с другими системами.

Особенности:

Автоматизированный вход: Скрипт автоматизирует процесс входа на сайт "https://journal.top-academy.ru" с использованием предоставленных учетных данных (имя пользователя и пароль).
Навигация: После входа скрипт переходит по определенному URL ("https://journal.top-academy.ru/ru/main/library/page/index/5"), где предполагается выполнение задачи парсинга.
Стабильность: Скрипт использует механизмы явного ожидания с использованием WebDriverWait из Selenium, чтобы убедиться, что элементы загружены перед взаимодействием с ними. Это повышает стабильность процесса скрапинга.
Обработка ошибок: В скрипте предусмотрена обработка ошибок для ситуаций, когда перенаправление на страницу парсинга не удается. Он пытается несколько раз перейти на страницу парсинга, прежде чем завершить цикл, если перенаправление не удалось.
Извлечение данных: Скрипт находит блоки с видео на странице парсинга и извлекает ссылки на iframe из этих блоков. Извлеченные ссылки затем сохраняются в текстовом файле "iframe_links.txt".
Грациозное завершение: По завершении извлечения данных скрипт корректно закрывает экземпляр веб-браузера.
Использование:

Убедитесь, что на вашей системе установлен Python.
Установите необходимые пакеты, выполнив команду: pip install selenium.
Скачайте соответствующую версию ChromeDriver, соответствующую версии вашего браузера Chrome, и поместите файл в ту же директорию, что и скрипт.
Измените переменную parse_url на желаемый URL для парсинга.
Предоставьте ваши учетные данные, обновив строки user_field.send_keys() и password_field.send_keys().
Запустите скрипт.
Извлеченные ссылки на iframe будут сохранены в файле "iframe_links.txt" в той же директории, что и скрипт.
Примечание:

Этот скрипт может потребовать настройки, если структура целевого сайта изменится или если будут внесены обновления в библиотеку Selenium.
Автоматизированный скрапинг должен выполняться ответственно и этично, с соблюдением условий использования сайта и рекомендаций по скрапингу.
